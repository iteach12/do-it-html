CSS와 박스모델

블록 레벨의 요소는 한 줄을 혼자 차지한다(너비가 100%라는 소리). 요소의 왼쪽이나 오른쪽에 다른 요소가 올 수 없다. 너비, 마진, 패딩 등을 이용하여 크기나 위치를 조정한다는 얘기는 블록 레벨이란 소리.

인라인 레벨의 요소는 줄을 차지하지 않는다. 콘텐츠만큼만 자리를 차지한다. 여러 개의 인라인 레벨 요소를 한 줄에 표시 가능.

종류 해당 태그
블록 레벨 태그 <p>, <h1>~ <h6>, <ul>, <ol>, <div>, <blockquote>, <form>, <hr>, <table>, <fieldset>, <address>
인라인 레벨 태그 <img>, <object>, <br>, <sub>, <sup>, <span>, <input>, <textarea>, <label>, <button>

none 속성 값

이 속성 값을 지정하면 해당 요소를 화면에 아예 표시하지 않는다. visibility:hidden도 비슷한 역할을 하는데, visibility 속성은 감추기만하고 자리는 차지하지만 none는 자리조차 없어진다 (안드로이드의 gone 이구만?)

border-width 속성 - 테두리 두께 지정하기

테두리 두께가 있어야 화면에 등장함. 지정하지 않으면 두께가 없기 때문에 어떤 스타일이든 화면에 표시될 수가 없겠지.

thin | medium | thick 세 가지 옵션 + 크기고정 방식

방향을 지정하지 않으면 border-width -> 네 방향 모두 같은 두께. border-top-width -> top border의 두께만 조절.
border-width:1px 2px 3px 4px; 는 차례대로 위, 오른쪽, 아래, 왼쪽 시계방향으로 적용됨.

border-top-width, border-bottom-width, border-left-width, border-right-width 가 있음.
