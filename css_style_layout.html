CSS와 박스모델

블록 레벨의 요소는 한 줄을 혼자 차지한다(너비가 100%라는 소리). 요소의 왼쪽이나 오른쪽에 다른 요소가 올 수 없다. 너비, 마진, 패딩 등을 이용하여 크기나 위치를 조정한다는 얘기는 블록 레벨이란 소리.

인라인 레벨의 요소는 줄을 차지하지 않는다. 콘텐츠만큼만 자리를 차지한다. 여러 개의 인라인 레벨 요소를 한 줄에 표시 가능.

종류 해당 태그
블록 레벨 태그 <p>, <h1>~ <h6>, <ul>, <ol>, <div>, <blockquote>, <form>, <hr>, <table>, <fieldset>, <address>
인라인 레벨 태그 <img>, <object>, <br>, <sub>, <sup>, <span>, <input>, <textarea>, <label>, <button>
