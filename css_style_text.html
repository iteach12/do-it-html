글꼴 관련 스타일

font-familly 속성 - 기본 글꼴;

글자의 모양새. 내용에 알맞은 글꼴과 글자 크기, 굵기와 이탤릭체 여부 등에 대한 내용. 

      
        font-family:<글꼴 이름>[,<글꼴 이름>, <글꼴 이름>]

이는 컴퓨터에 설치된 기본 글꼴을 이용해 화면에 출력하는 방식이다. 뒤의 대괄호는 사용자에게 해당 글꼴이 없는 경우를 대비하여 2순위, 3순위... 등의 글꼴을 지정해 주는 것이다.

font-size 속성 - 글자 크기 조정

속성 값 설명
절대 크기 브라우저에서 지정한 글자 크기입니다. 사용할 수 있는 값은 xx-small | x-small | small | medium | larger | x-large | xx-large 입니다.
상대크기 부모 요소의 글자 크기(font-size)를 기준으로 더 크게 표시하거나 더 작게 표시합니다. 사용할 수 있는 값은 larger | smaller입니다.
크기 브라우저와 상관없이 글자 크기를 직접 지정합니다.
백분율 부모 요소의 글자 크기를 기준으로 해당하는 %를 계산해 표시합니다.( % 기호가 붙어야 함. )
em 해당 글꼴의 대문자 M의 너비를 기준으로 크기를 조절합니다. 1em = 16px
ex x-height(엑스 하이트). 해당 글꼴의 소문자 x의 높이를 기준으로 크기를 조절합니다.
px 픽셀. 모니터에 따라 상대적 크기가 됩니다.
px단위를 사용하면 폰트 크기가 고정되기 때문에 창이 작은 모바일 기기로 볼 때도 같은 크기로 표현됩니다. 따라서 모바일까지 고려한다면 px보다는 em이 낫다.
pt 포인트. 일반 문서에서 많이 사용하는 단위입니다.

em의 사용법
부모요소 font-size:20px;

부모의 font-size가 20px일 때, 자식 요소의 font-size가 1em이라면 그대로 20px에 맞춰서 출력되고, 2em은 두 배, 3em은 세 배 등등

color 속성

color 속성은 글자의 색깔을 나타낼 때 사용한다. rgb, 16진수, hsl 또는 색상 자체의 이름으로 나타낼 수 있다. 

      
        h1{color:tgb(0,200,0)} 
        h2{color:blue;}
        .accent{
          color:#ff0000; 
        }

text-decoration 속성

text-decoration
속성 값 설명
none 밑줄을 표시하지 않는다.
underline 밑줄을 표시한다.
overline 영역 위로 선을 그린다.
line-through 영역을 가로지르는 선(취소 선)을 그린다.
text-decoration 예시

없음. a태그의 밑줄 지울 때 / text-decoration: none;

밑줄 / text-decoration: underline;

취소선 / text-decoration: line-through;

윗줄 / text-decoration: overline;

text-transform 속성

속성 값 설명
none 변환하지 않는다.
capitalize 시작하는 첫 번째 글자를 대문자로 변환한다.
uppercase 모든 글자를 대문자로 변환한다.
lowercase 모든 글자를 소문자로 변환한다.
fullwidth 가능한 모든 문자를 전각 문자로 변환한다.
text-transform 예시

Hello to the world.
첫 글자를 대문자로.

Hello to the world.
전부다 대문자로.

Hello to the world.
전부다 소문자로.

Hello to the world.
전부다 전각자로. // ??
