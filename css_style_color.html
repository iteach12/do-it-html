background-color 속성

※ 스타일 상속에 의해서 부모 스타일은 자식 스타일로 똑같이 적용되지만 예외적으로 background-color값은 상속되지 않음.

background-clip 속성

박스 모델 관점에서 배경 적용 범위를 조절할 수 있다. 기본값은 border-box

속성 값 설명
border-box 박스 모델의 가장 외곽인 테두리(border)까지 적용됨
padding-box 박스 모델에서 테두리를 뺀 페딩 범위까지 적용됨.
content-box 박스 모델에서 내용 부분에만 적용됨.

background-image 속성

url을 사용하여 절대, 상대 경로를 넣어준다. 화면 크기보다 작은 이미지인 경우 자동으로 반복하여 넣어준다.

background-repeat 속성

속성 값 설명
repeat 브라우저 화면에 가득찰 때까지 가로와 세로로 반복.
repeat-x 브라우저 창 너비와 같아질 때까지 가로로 반복.
repeat-y 브라우저 창 높이와 같아질 때까지 세로로 반복.
no-repeat 반복 없음.

repeat

repeat-x

repeat-y

no-repeat

background-size 속성

이미지가 화면보다 더 큰 경우 이미지를 여러 크기로 조절할 수 있다.

속성 값 설명
auto 원래 배경 이미지 크기 만큼 표시됨. 기본*
contain 요소 안에 배경 이미지가 다 들어오도록 이미지를 확대/축소 함.
cover 배경 이미지로 요소를 모두 덮도록 이미지를 확대/축소함.
크기값 너비, 높이 값을 지정. 둘 중 하나만 지정했을 땐 원래 크기를 기준으로 축소/확대하여 표시.
백분율 백분율로 지정.

contain

background-position 속성

배경 이미지가 표시되는 위치를 조절할 수 있다.

수평, 수직 두 가지 수치를 넣어줘야 하는데, 하나만 넣는다면 수직위치 값은 자동으로 50% 또는 center로 간주된다.

background-position: left

background-position: right

background-origin 속성

background-position을 이용해서 이미지를 배치할 때 배치할 기준이 필요함. (background-clip 처럼) 이 기준을 origin으로 정함. 속성 값의 의미는 clip와 똑같음.

background-attachment 속성

배경 이미지 고정하기. scroll과 fixed 속성을 활용.

속성 값 설명
scroll 화면 스크롤과 함께 배경 이미지도 스크롤 됨. 기본값
fixed 이미지 고정
