아래는 list관련 tag입니다.

unordered list(ol) + li

순서가 필요하지 않은 목록을 만들 때 ul 태그 안에 li태그를 넣는다.

내가 가장 좋아하는 것

  • computer
  • coding
  • book
  • exercise

ordered list (ol) + li

순서가 있는 목록. type을 적용해서 원하는 형태로 표시할수 있다.

  1. 아무런
  2. 타입을
  3. 적용하지
  4. 않으면
  5. 숫자로 나오는구나.

가장 기본은 숫자임. a, A, i, I 와 같은 기호들이 가능하다. 적용 방법은?

  • 1일차
    1. 타입을 a로 바꾸면
    2. 이렇게 나옵니다.
  • 2일차
    1. 타입을 바꾸고
    2. 시작 숫자도 바꿀 수 있습니다.
