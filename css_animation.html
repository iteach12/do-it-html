transform

사용자의 동작에 반응해 텍스트나 이미지 등을 움직이게 함.

2차원, 3차원 변형

  • CSS 스타일 시트 소스만으로 이미지를 변형할 수 있다.
  • 2차원 변형은 수평이나 수직으로 이동, 회전시키는 것을 말한다.
  • x축은 오른쪽으로 갈 수록 값이 커지고, y축은 아래로 갈 수록 값이 커진다.
  • 3차원 변형은 x축과 y축에 원근감 z축을 추가한다.
  • z축은 앞뒤로 이동하는데 보는 사람 쪽으로 다가올수록 값이 더 커진다.

transform과 변형 함수

2차원 변형 함수
변형 함수 설명
translate(tx,ty) 지정한 크기만큼 x축과 y축으로 이동
translateX(tx) 지정한 크기만큼 x축으로 이동
translateY(tx) 지정한 크기만큼 y축으로 이동
scale(sx,sy) 지정한 크기만큼 x축과 y축으로 확대/축소
scaleX(sx) 지정한 크기만큼 x축으로 확대/축소
scaleY(sy) 지정한 크기만큼 y축으로 확대/축소
rotate(각도) 지정한 크기만큼 회전
skew(ax,ay) 지정한 크기만큼 x축과 y축으로 왜곡(단위:deg, rad, grad)
skewX(ax) 지정한 크기만큼 x축으로 왜곡
skewY(ay) 지정한 크기만큼 y축으로 왜곡
  • 3차원 변형함수 = 2차원 변형함수 + z축
  • perspective(원근법) 속성을 사용해야 입체감을 느낄 수 있다.
  • perspective 속성은 원근감을 주려는 div나 img의 부모요소에 적용해야 효과를 줄 수 있었다.

animation

속성 설명
@keyframes 애니매이션이 바뀌는 지점을 설정
animation-delay 애니매이션 지연 시간
animation-direction 애니매이션 종류 후, 처음부터 시작할지, 역방향으로 진행할지 지정
animation-duration 애니매이션 실행 시간
animation-fill-mode 애니매이션이 종료 혹은 지연되어 실행되지 않고 있을 때 요소의 스타일을 지정
animation-iteration-count 애니매이션 반복 횟수
animation-name keyframes로 설정해 놓은 중간 상태의 이름을 지정
animation-play-state 애니매이션을 멈추거나 다시 시작
animation-timing-function 애니매이션 속도 곡선을 지정
