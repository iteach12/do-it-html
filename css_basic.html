스타일과 스타일 시트

CSS는 텍스트 생삭이나 크기, 이미지 크기나 위치, 표 색상, 배치 방법 등 웹 문서의 디자인 요소를 담당한다.

다양한 기기에 맞게 탄력적으로 바뀌는 문서를 만들 수 있다.

내부 스타일 시트 -> head 태그 안에 정의
외부 스타일 시트 -> link로 css파일 연결
css파일 임포트하기 -> @import url(/css/css_basic.css); 와 같이 사용

주요 선택자

  1. * 전체 선택자
    1. 모든 요소에 적용할 때 사용
  2. 태그 선택자
    1. 특정 태그가 쓰인 모든 요소에 스타일 적용
  3. . 클래스 선택자
    1. 클래스 이름은 태그 이름과 겹치면 안됨.
    2. 클래스 이름 앞에 .을 붙여야 함.
    3. 특정 태그에만 클래스 선택자를 적용하고 싶을 때는 h2.accent처럼 태그에 이어서 .을 찍고 클래스 이름을 적어준다.
  4. # ID 선택자
    1. #을 붙여서 ID임을 나타내 준다.
  5. space 공백을 넣으면
    1. div img 와 같이 선택자 사이에 공백을 넣으면
    2. div 요소아래에 있는 모든 img 요소에 적용이 된다. 이는 '>'와 다르다.
  6. '>' 꺽쇠를 넣으면
    1. div>img 와 같이 선택자 사이에 '>'를 넣으면
    2. div 바로 아래에 있는 img에만 적용이 된다.
    3. 참고자료

캐스케이딩

CSS에서 C는 캐스캐이딩의 약자로 위에서 아래로 흐르는 스타일 시트라는 뜻이다.

  • 스타일 우선순위 : 스타일 규칙의 중요도, 적용 범위에 따라 우선순위가 결정되고 그 우선순위에 따라 위에서 아래로 스타일이 적용된다.
  • 스타일 상속 : 태그들의 포함 관계에 따라 부모 요소의 스타일을 자식 요소로, 위에서 아래로 전달
스타일 시트의 우선순위

스타일 적용 범위가 좁을수록 즉, 정확히 필요한 요소에만 적용할 스타일일수록 우선 순위가 높다.

  1. 인라인 스타일
  2. id 스타일
  3. 클래스 스타일
  4. 태그 스타일
  5. !important 를 붙이면 가장 강력한 순위를 가짐.

스타일 상속

태그들의 포함 관계 중 포함하는 태그를 부모 요소, 포함된 태그를 자식요소라고 함.

자식 요소에서 별도로 스타일을 지정하지 않으면 부모 요소에 있는 스타일 속성들이 자식 요소로 전달되는데 이것을 '스타일 상속'이라고 함.

부모 요소의 모든 속성이 자식 요소에게 상속되는 것은 아니다.
(글자 색은 자식 요소로 상속되지만 배경 이미지나 배경 색은 상속되지 않는다.)
