img Tag

src 속성으로 이미지 파일 경로 지정하기

이미지 파일의 경로는 웹 문서 파일의 위치를 기준으로 정해진다. 웹 문서와 이미지가 같은 경로에 있다면 이미지 파일의 이름만으로 불러올 수 있다.

style 태그를 이용하여 width 와 height 를 한 번에 넣어 주는 것이 좋다. 왜냐하면 style css에서 중복으로 효과를 주는 것을 예방하기 위해서.

float는 텍스트의 left나 right에 img를 위치시킬 수 있는 속성. 그런데 어떤 경우 사용하는지 잘 이해가 안간다.

속성 설명
href 링크한 문서나 사이트의 주소를 입력합니다.

hreflang 링크한 문서의 언어를 지정합니다.
type 링크한 문서의 파일 유형을 알려줍니다.
