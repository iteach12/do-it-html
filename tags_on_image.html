img Tag

src 속성으로 이미지 파일 경로 지정하기

이미지 파일의 경로는 웹 문서 파일의 위치를 기준으로 정해진다. 웹 문서와 이미지가 같은 경로에 있다면 이미지 파일의 이름만으로 불러올 수 있다.

style 태그를 이용하여 width 와 height 를 한 번에 넣어 주는 것이 좋다. 왜냐하면 style css에서 중복으로 효과를 주는 것을 예방하기 위해서.

float는 텍스트의 left나 right에 img를 위치시킬 수 있는 속성. 그런데 어떤 경우 사용하는지 잘 이해가 안간다.

앵커 만들기

앵커는 한 페이지 내부에서 이동할 수 있는 기능을 말한다. 목차를 누르면 해당하는 위치로 이동하듯.

목적지에 id를 생성해준다. id="goal__move" 와 같이..
앵커는 a 태그를 이용하는데, href 태그에 #을 넣고 목적지의 id를 넣어주면 된다.

image map

하나의 이미지에 하나의 링크?! image map을 사용하면 하나의 이미지에 여러개의 링크를 걸 수 있다.

이미지 상의 클릭 위치를 다르게 하여 각각의 위치마다 다른 링크를 부여.

map 태그를 이용해서 image map을 일단 만든다. 맵 이름도 지정해주고 나서, 맵 안에 area 태그로 영역을 생성해준다.

area태그의 속성들
속성 설명
alt 대체 텍스트를 지정합니다.
coords 링크로 사용할 영역을 시작 좌표와 끝 좌표를 이용해 지정합니다.
download 링크를 클릭했을 때 링크 문서를 다운로드 합니다.
href 링크 문서(사이트) 경로를 지정합니다.
media 링크 문서(사이트)를 어떤 미디어에 최적화시킬지 지정합니다.
rel 현재 문서와 링크 문서 사이의 관계를 지정합니다. 속성 값 lternate, bookmark, help, license, next, nofollow, noreferer, prefetch, prey, search, tag
shape 링크로 사용할 영역의 형태를 지정합니다. default, rect, circle, poly
target 링크를 표시할 대상을 지정합니다. _blank, _parent, _self, _top, 프레임 이름
type 링크 문서의 미디어 유형을 지정합니다.
