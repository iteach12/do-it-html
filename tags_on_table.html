기본적인 표 만들기

  1. table 태그를 이용해서 전체적인 윤곽을 만든다.
  2. tr 태그로 행을 만든다.
  3. td 태그로 각 행마다 셀을 만든다.
  4. 각 셀에 들어갈 내용은 td 태그에 넣는다.
  5. th 태그를 이용하여 행의 제목을 붙여 준다.
  6. colespan 태그, rowspan 태그를 활용하여 행이나 열을 합친다.

일부 셀 합치기

colspan 태그, rowspan 태그를 활용하여 셀 병합하기

caption 태그를 사용하여 중앙정렬된 표 제목 달기. caption 태그 안에는 여러 줄이나 태그들을 혼합하여 사용할 수 있다.

figcaption 태그는 figure와 caption을 합친 태그다.

figure 태그는 겉을 감싸고, figcaption 태그로 figure의 제목(caption)을 붙여주는구만. 표 말고도 이용할 수 있다고 함.

rowspan 태그와 colspan 태그는 같은 종류 안에서만 사용할 수 있다. 예를 들면, tbody 안에서만, 또는 tfoot 안에서만 셀 병합이 가능하다. tbody에서 시작해서 tfoot의 셀을 병합할 순 없다.
