핵심요소

  • 자식 엘리멘트에는 어떤 것도 적지 말아야 한다. only 부모 요소에만 적는다. 자식을 움직이고 싶다면 부모에게 적는다. 부모의 display를 flex로 설정한다.
  • justify-content : main axis(주축)에 적용된다. align-items : cross axis(교차축)에 적용된다.
  • justify-content가 수평이라고 말할 수 없다. 왜냐하면 main axis의 기본은 수평이지만 이를 수직으로 바꿀 수도 있기 때문. 마찬가지로 align-items가 수직이라고할 순 없다. cross axis의 기본이 수직이지만 이를 수평으로 바꿀 수도 있기 때문이다.
  • height:100vh는 ViewPort의 height를 100%로 주었다는 의미.
  • flex 컨테이너 안의 자식 요소도 flex 컨테이너가 될 수 있다. 단, flex 속성이 적용되는 범위는 flex 컨테이너의 자식까지만이다. 부모의 부모가 flex인 것은 손자에게 전해지지 않는다. 부모 요소가 flex여야만 자식 요소가 그 영향을 받는다.
  • flex-wrap:nowrap 기본값. flex는 width를 초기값으로 생각. nowrap인 경우 요소들을 한 줄에 위치시키기 위해서 width값을 무시 하게 된다. 반면, flex-wrap:wrap로 바꾸면 요소들을 여러 줄에 위치시킬 수 있게 되고 width값도 유지할 수 있게 된다.
