연결 선택자

연결 선택자는 '콤비네이션 선택자', '콤비네이션 셀렉터' 라고도 함.

하위 선택자(descendant selector) - 지정한 모든 하위 요소에 스타일 적용하기

      
        section p {color:blue;}
        상위요소 하위요소
        스페이스로 구분

부모 요소에 포함된 하위 요소 모두에 스타일이 적용됨. '자손 선택자'라고도 함. 자식 요소 뿐 아니라 그 안의 자식의 손자, 손자의 손자 요소 등 모든 하위 요소에 적용.

자식 선택자(child selector) - 자식 요소에만 스타일 적용하기


      section > p {color:blue;}
      부모요소 > 자식요소
      부등호로 구분

자식 선택자는 부모 요소의 바로 아래에 있는 요소, 즉 자식 요소에만 스타일 적용.

인접 형제 선택자(adjacent selector) - 가장 가까운 형제 요소에 스타일 적용하기


      h1 + p {text-decoration:underline;}
      h1과 형제인 p 중에서 첫 번째 p에만 스타일 적용

h1 태그의 형제들 중 가장 먼저 오는 요소에만 적용한다.

형제 선택자(sibling selector) - 형제 요소에 스타일 적용하기

인접 형제 선택자와 달리 모든 형제 요소에 적용됨. '~'틸드를 사용하여 구분. 


      h1 ~ p {text-decoration:underline;}

속성 선택자

[속성]선택자 - 지정한 속성에 스타일 적용하기

특정한 속성을 사용한 요소에 스타일을 적용하는 선택자. 대괄호 사이에 속성을 지정 


      a[href]{
        background:yellow;
      }

[속성=값]선택자 - 특정 값을 갖는 속성에 스타일 적용하기

속성 형식에서 더 나아가 일치하는 값을 가진 경우에만 스타일을 적용할 수도 있다. 


      a[target="_blank"]{
        padding-right:30px;
        background:url(images/newwindow.png) no-repeat center right;
      }

[속성 ~= 값] 선택자 - 여러 값 중 특정 값이 포함된 속성에 스타일 적용하기

[속성=값]의 경우 정확히 일치하는 요소에만 적용되나, 틸드(~)와 함께 사용한 선택자 의 경우에는 해당 값이 포함된 요소를 선택하여 적용해 준다. 하지만, 값이 다르면 적용이 안되니 유의할 것. 

      
        [class~="button"]{
          border:2px solid black;
          box-shadow:rgba(0, 0, 0, 0.4) 5px 5px;
        }

속성 선택자 - 정리

기본형 스타일 적용 경우 예시 - 선택자 예시 - 적용되는 모습
[속성] 지정한 '속성' 일 때 [href]
[속성=값] '값'에 일치할 때 [target="_blank"]
[속성 ~= 값] 여러 값들 중 특정 '값'이 포함되어 있을 때(단어별) [class~="button"]
[속성 |= 값] '값'이 포함되어 있을 때(하이픈 포함 단어별) [class |= "us"]
[속성 ^= 값] '값'으로 시작할 때 [title ^= "eng"]
[속성 $= 값] '값'으로 끝날 때 [href $= "xls"]
[속성 *= 값] '값'이 속성 값의 일부일 때 [href *= "w3"]

가상 클래스와 가상 요소

사용자 동작에 반응하는 가상 클래스

테스트 확인하기

가상 선택자를 링크와 관련해 사용할 때는 순서가 중요하다. :link, :visited, :hover, :active가 순서대로 지정되어야 정상 작동한다.

UI 요소 상태에 따른 가상 클래스

  1. :enabled와 :disabled - 해당 요쇼가 사용 가능한 상태일 때의 스타일은 enabled, 불가능한 상태일 때의 스타일은 disabled로 지정.
  2. :checked - 라디오나 체크 박스에서 해당 항목을 선택했을 때

구조 가상 클래스

  1. :root - 문서 전체에 적용하기
  2. :nth-child(n)과 :nth-last-child(n) - 자식 요소의 위치에 따라 스타일 적용하기
  3. :nth-of-type(n), :nth-last-of-type(n) - 특정 태그 위치에 스타일 적용하기
