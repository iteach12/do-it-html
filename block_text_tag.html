Block level elements

  • 블록 수준 요소는 항상 새 줄에서 시작합니다.(A block-level element always starts on a new line.)
  • 블록 수준 요소는 항상 사용 가능한 전체 너비를 차지합니다.(가능한 왼쪽과 오른쪽으로 늘어남.) A block-level element always takes up the full width available (stretches out to the left and right as far as it can).
  • 블록 수준 요소에는 위쪽 및 아래쪽 margin이 있는 반면 인라인 요소에는 없습니다. A block level element has a top and a bottom margin, whereas an inline element does not.

줄바꿈은 br. 닫는 기호는 없다. break의 줄임말.

hr 수평선
